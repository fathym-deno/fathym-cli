# Fathym CLI Documentation

Welcome to the Fathym CLI documentation. This CLI framework enables you to build, scaffold, and distribute command-line interfaces using Deno.

### API Reference

Command documentation with arguments, flags, and examples:

| Command | Description |
|---------|-------------|
| [ftm init](./api/init.mdx) | Scaffold a new CLI project |
| [ftm build](./api/build.mdx) | Prepare static build artifacts |
| [ftm compile](./api/compile.mdx) | Compile to native binary |
| [ftm run](./api/run.mdx) | Execute command in development mode |
| [ftm test](./api/test.mdx) | Run intent tests |
| [ftm install](./api/install.mdx) | Install compiled binary to PATH |
| [ftm cli config](./api/config.mdx) | Manage user configuration |
| [ftm cli release](./api/release.mdx) | Build, compile, and package for all platforms |
