# Getting Started

Create your first CLI in 5 minutes.

## Prerequisites

- [Deno](https://deno.land/) installed (v1.40+)

## Quick Start

### 1. Initialize a New Project

```bash
ftm init my-cli
cd my-cli
```

This creates a complete CLI project with sample commands.

### 3. Run a Command

```bash
# Development mode (no compilation)
ftm run hello

# Output: Hello, World!
```

```bash
# With arguments
ftm run hello Alice

# Output: Hello, Alice!
```

```bash
# With flags
ftm run hello Alice --loud

# Output: HELLO, ALICE!
```

### 4. Run Tests

```bash
ftm test
```

### 5. Build and Compile

```bash
# Prepare static build
ftm build

# Compile to native binary
ftm compile
```

### 6. Run the Compiled Binary

```bash
# Unix
./dist/my-cli hello

# Windows
.\dist\my-cli.exe hello
```

### 7. Install Globally (Optional)

```bash
ftm install --useHome --to=.bin
```

Add `~/.bin` to your PATH to use globally.

## Understanding the Sample Commands

### hello.ts (Fluent API)

```typescript
export default Command('hello', 'Prints a friendly greeting.')
  .Args(HelloArgsSchema)
  .Flags(HelloFlagsSchema)
  .Params(HelloParams)
  .Run(({ Params, Log }) => {
    const name = Params.Name;
    const message = Params.Loud
      ? `HELLO, ${name.toUpperCase()}!`
      : `Hello, ${name}.`;

    if (Params.DryRun) {
      Log.Info(`ðŸ›‘ Dry run: "${message}" would have been printed.`);
    } else {
      Log.Info(message);
    }
  });
```
